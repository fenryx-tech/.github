# Fenryx GitHub 公開素材庫

## 概述

本目錄存放所有可在 GitHub 上公開使用的 Fenryx 品牌素材，包括 Logo、Banner 等圖片資源。

## 資料夾結構

```
profile/assets/
├── README.md          # 本文件
├── logo/              # Logo 相關素材
├── banner/            # Banner 相關素材
└── social/            # 社群媒體素材
    └── linkedin/      # LinkedIn 專用素材
```

## 使用方式

詳細的使用說明請參考：
- [素材使用規範](../ASSETS_USAGE_GUIDE.md)
- [素材資料夾結構規劃](../ASSETS_STRUCTURE.md)

## 引用路徑

所有素材的引用基礎路徑為：
```
https://raw.githubusercontent.com/fenryx-tech/.github/main/profile/assets/
```

### 範例

- Logo: `https://raw.githubusercontent.com/fenryx-tech/.github/main/profile/assets/logo/logo-128.png`
- Banner: `https://raw.githubusercontent.com/fenryx-tech/.github/main/profile/assets/banner/github-banner.png`

## 檔案命名規範

1. **小寫字母**：所有檔案名稱使用小寫字母
2. **連字號分隔**：使用連字號 `-` 而非底線 `_`
3. **描述性名稱**：檔案名稱應清楚描述內容
4. **尺寸標示**：Logo 檔案應包含尺寸資訊（如 `logo-128.png`）

## 素材說明

所有素材按照用途分類存放，方便查找和使用。如需更多資訊，請參考相關文件。
