# .github

This repository contains public assets and organization profile for Fenryx.

## Structure

```
.github/
├── profile/
│   ├── README.md           # Organization profile page
│   └── assets/             # Public brand assets
│       ├── logo/           # Logo files (various sizes)
│       ├── banner/         # Banner images
│       └── social/         # Social media assets
└── README.md               # This file
```

## Usage

Assets in this repository are publicly accessible and can be referenced using:

```
https://raw.githubusercontent.com/fenryx-tech/.github/main/profile/assets/[category]/[filename]
```

Example:
```markdown
![Fenryx Logo](https://raw.githubusercontent.com/fenryx-tech/.github/main/profile/assets/logo/logo-128.png)
```
